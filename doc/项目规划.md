# sts2kernel 项目发展规划

## 项目概述

**项目名称：** sts2kernel - 杀戮尖塔2 无GUI单人纯内核项目  
**项目位置：** `C:\ZZZZZZ\03_python\34_mySts2Kernel\sts2kernel\kernel`  
**项目目标：** 将杀戮尖塔2游戏从依赖Godot的GUI版本重构为纯C#内核版本，移除所有图形界面依赖，保留核心游戏逻辑，实现可确定性重放的游戏运行。

## 风险与挑战

### 1. RNG一致性
**风险：** 不同平台或.NET版本的`System.Random`实现可能不同  
**解决方案：** 实现自定义的确定性RNG，不依赖`System.Random`

### 2. 状态复杂性
**风险：** 游戏状态过于复杂，难以维护  
**解决方案：** 使用状态机模式，清晰定义状态转换

### 3. 性能问题
**风险：** 纯C#实现可能比Godot版本慢  
**解决方案：** 优化热点代码，使用对象池

### 4. 兼容性
**风险：** 与原版游戏行为不一致  
**解决方案：** 建立完整的测试套件，对比原版行为
