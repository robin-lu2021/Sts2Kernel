Welcome to the first BIG post-launch patch, Slayers!! This one has a huge balance pass, the main aim of which is to make infinites harder to achieve. We're also introducing a Phobia Mode with bespoke assets that should mitigate various creepy phobia concerns. Speaking of visuals, here are some other major highlights: some snazzy new portrait art for various cards and cool new character-specific VFX for when you gain energy!

Reminder that [b]this patch is exclusive to the beta branch branch for now[/b], which you can access via the instructions outlined in the first beta patch notes. It will come to the main branch later, likely with some more tweaks!
